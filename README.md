# DoGit
###### 개인 프로젝트 (진행중)

### Github Repository별로 Todo List를 만들 수 있는 App

<br>

### 만들게 된 이유?
Github의 여러 Repository에 커밋을 하다보니 issue를 작성하는 것보다 가벼운
Github Repository 만의 Todo를 사용하고 싶어 제작하게 되었습니다.
